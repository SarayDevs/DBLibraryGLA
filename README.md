# DBLibraryGLA
This project is a library database for Gimnasio Los Almendros, built from scratch. It replaces Excel for managing loans, viewing book information, and adding new entries, optimizing these processes. Developed using PHP, MySQL, JavaScript, HTML, and CSS.

# Bienvenido a mi repositorio / Welcome to my repository

Este es el proyecto de biblioteca, donde se gestionan libros y usuarios. <br>
This is a library project where books and users are managed.

## 📌 Funcionalidades / Features
- 📚 Gestión de libros / Book management
- 👤 Gestión de usuarios / User management
- 📥 Agregar libros de manera más eficiente / Add books more efficiently
- 🔄 Verificar y agregar préstamos / Verify and add loans
- ❌ Eliminar libros / Delete books
- 📊 Verificar actividad de cada libro / Check each book's activity
- 🔍 Filtrar por actividad / Filter by activity
- ✏️ Actualizar información del libro / Update book information

## 🚀 Cómo instalar / How to install
- Clona el repositorio / Clone the repository.
- Instala las dependencias / Install the dependencies.
